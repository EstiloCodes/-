1. 현재의 학습 방향과 단기 목표
현재 가장 집중하고 있는 부분은 컴퓨터공학의 뼈대가 되는 기초를 확실히 다지는 것입니다. C, 자바(Java), 파이썬(Python)을 학습하며 언어별 특성과 객체지향 프로그래밍의 원리를 체화하고 있습니다. 또한 디지털 논리회로와 선형대수학 같은 과목을 통해 소프트웨어가 하드웨어 위에서 어떻게 동작하고 연산되는지 근본적인 이해도를 높이고 있습니다. 올해의 핵심 목표는 자료구조(Data Structures)를 완벽히 이해하여 효율적인 알고리즘 설계와 코드 작성의 기반을 마련하는 것입니다.

2. 중장기 목표 및 실무 역량 강화
단순히 작동만 하는 코드를 짜는 것을 넘어, '메모리 관리'와 '시스템 최적화'를 깊이 있게 다룰 수 있는 엔지니어로 성장하고 싶습니다. 이를 위해 학년이 올라가면 선배들의 졸업 작품이나 우수한 오픈소스 코드를 적극적으로 분석하며, 실제 프로젝트에서 코드가 어떻게 구조화되는지 실무적인 감각을 익힐 계획입니다. 또한, GitHub을 활용해 제가 공부한 내용과 개인 프로젝트 진행 상황을 꾸준히 기록하여 저만의 기술 블로그이자 포트폴리오로 만들어가겠습니다.

3. 최종 진로 방향
최종적으로는 시스템 자원을 효율적으로 관리하고 최적화할 수 있는 백엔드 개발자(Backend Developer) 혹은 시스템 소프트웨어 엔지니어가 되는 것이 목표입니다. 단순히 주어진 기술을 사용하는 것에 그치지 않고, 항상 원리를 깊이 파고드는 자기주도적인 개발자가 되겠습니다.
